*************
How the system work:
1. Register with email + password
2. Scan QR with Authenticator
3. Login
4. Enter OTP
5. Check biometric
6. Access dashboard


*************
How to run backend:
cd backend
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload


*************
How to run frontend:
cd frontend
npm install
npm run dev
